# Description

This page provides a comprehensive illustrated reference guide to golf terminology organized into 13 major categories.
It covers equipment terms (club types and parts), technique terms (setup, swing mechanics, and common errors), ball flight patterns, and shot variations.
The resource explains scoring systems including handicap calculations, various game formats, and named score types from double bogey to albatross.
Course element definitions help golfers understand different areas of play including teeing grounds, fairways, rough, bunkers, and hazards.
This serves as a complete glossary for golfers at all skill levels to understand the language and technical aspects of the game.
