# Locking Your Left Knee at Impact

This swing error occurs when the left leg gradually straightens during the downswing, resulting in a completely straight leg with no knee flex at impact.
The main consequence is that it changes hip angle and direction, pulling hips left and creating an outside-in club path that produces left-to-right spin.
