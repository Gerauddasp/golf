# Golf Setup - Description

This page provides comprehensive illustrated definitions of golf setup fundamentals and related terminology. Setup encompasses all aspects of body positioning relative to the ball, club, and target before beginning a swing. The guide covers six core setup components (back tilt, knee flex, arms/hands position, feet position, grip, and ball position) along with related concepts including aim, alignment, line, and target line. Each element is explained with detailed illustrations showing proper positioning and technique.
