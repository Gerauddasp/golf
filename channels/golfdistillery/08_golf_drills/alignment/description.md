# Alignment Drill - Description

Proper golf alignment is the foundation of a consistent swing and game. This lesson teaches how to position your feet, knees, hips, and shoulders parallel to the target line while keeping the clubface square. Learn to identify closed, open, and square stances using simple club placement tests. Practice methods include using alignment sticks, getting external feedback, and establishing reference points for your pre-shot routine. Master this fundamental setup element to ensure all other swing improvements can take effect.
