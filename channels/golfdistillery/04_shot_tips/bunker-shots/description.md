# Bunker Shots Overview

Comprehensive guide to executing golf shots from sand bunkers in nine different situations.
Covers regular greenside bunkers plus specialized scenarios including wet sand, plugged lies, and fried-egg lies.
Addresses upslope and downslope lies requiring specific setup adjustments to avoid common errors.
Includes techniques for both long and short greenside bunker shots with varying green distances.
Provides fundamentals for fairway bunkers where distance changes the approach fundamentally.
