This page explains how to fix push shots that start right of the target and continue straight without spin.
Push shots occur when the clubface is open at impact but square to an inside-out swing path.
The main fixes include checking alignment, strengthening grip, and correcting an inside takeaway.
Additional causes include blocking shots, locked right knee, excessive grip pressure, and improper ball position.
The page provides detailed diagrams and step-by-step solutions for each potential cause of pushed shots.
